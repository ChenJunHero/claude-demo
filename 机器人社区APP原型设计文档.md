## 1. APP导航结构

### 1.1 底部Tab导航

```
┌─────────────────────────────────────────────┐
│                                             │
│           [主要内容区域]                      │
│                                             │
│                                             │
└─────────────────────────────────────────────┘
┌─────────────────────────────────────────────┐
│  [🏠]      [🎮]      [+]      [🎉]      [👤]  │
│  首页      控制      发布      活动      我的   │
└─────────────────────────────────────────────┘
```

**Tab说明**：
- **首页**：社区内容流（推荐/关注/热门）
- **控制**：机器人控制中心（设备列表、遥控入口）
- **发布**：快速发布内容（中间凸起按钮）
- **活动**：运营活动、任务、排行榜
- **我的**：个人中心、设置

中性色
─────
- 深色文字：#1A1A1A
- 常规文字：#333333
- 次要文字：#666666
- 辅助文字：#999999
- 分割线：#EEEEEE
- 背景色：#F8F8F8
- 纯白：#FFFFFF

功能色
─────
- 点赞红：#FF2D55
- 收藏黄：#FFCC00
- 分享蓝：#0066FF
- 在线绿：#00CC66
- 离线灰：#CCCCCC

### 9.2 字体规范

```
iOS
─────
- 标题：PingFang SC Medium 18-24pt
- 正文：PingFang SC Regular 14-16pt
- 辅助：PingFang SC Light 12-13pt
- 数字：DIN Alternate Bold

Android
─────
- 标题：Noto Sans SC Medium 18-24sp
- 正文：Noto Sans SC Regular 14-16sp
- 辅助：Noto Sans SC Light 12-13sp
- 数字：Roboto Bold
```

### 9.3 间距规范

```
基础间距单位：4pt/dp

常用间距
─────
- 4pt:  最小间距
- 8pt:  内容内间距
- 12pt: 小组件间距
- 16pt: 卡片内边距
- 20pt: 模块间距
- 24pt: 大模块间距
- 32pt: 分组间距
```

### 9.4 圆角规范

```
- 2pt:  小标签
- 4pt:  按钮、输入框
- 8pt:  卡片
- 12pt: 大卡片
- 圆形: 头像、徽章
```

### 9.5 阴影规范

```
轻阴影（卡片）
─────
- iOS: shadow(color: rgba(0,0,0,0.08), offset: (0,2), blur: 8)
- Android: elevation: 2dp

中阴影（浮层）
─────
- iOS: shadow(color: rgba(0,0,0,0.12), offset: (0,4), blur: 12)
- Android: elevation: 4dp

重阴影（弹窗）
─────
- iOS: shadow(color: rgba(0,0,0,0.16), offset: (0,8), blur: 16)
- Android: elevation: 8dp
```
